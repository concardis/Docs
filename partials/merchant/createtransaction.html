Create new order

The Merchant can create different types of order depending of the Payment Method:
  • Create new Credit Card Order
  • Create new RatePAY Direct Debit Order
  • Create new RatePAY Invoice Order
  • Create new SEPA Direct Debit Order
 

Create new Credit Card Order

    • Click on „Plus“ icon on top right corner
    • New window will open to create a order
 
    • Select order Payment method type. (In this case Credit Card)
    • New input fields will appear for creating a credit card order.
    • First part are optional information input fields
      • MerchantOrder ID - Id of the order for merchant
      • Description  - Description of the order
      • Statement description - Statement description
  • Second part are input fields for credit card
  • Amount (Required) - Amount of the order
  • Currency (Required)  - Currency of the order. You can select from list of available currencies.
    Important!:
  • Card Number (Required) - Credit card number
  • Expiry Date (Required) - Expiry Date of the credit card ( mm/yy )
  • Security code (Required) - Security code of the Credit card ( 3 digit code on back of credit card)
  • Card Holder - Credit card holder
   
    • Third part are for adding additional information.
If you check on "Add Basket", it will open new section with basket additional fields. More about basket you can find in "How to manage Baskets". If you check on "Add Customer informattion", it will open new section with customer additional fields.
  • Last part are type of the transaction action. There are tree types of credit card transaction actions: Preauth, Debit and Credit.
  Important: By default, the "Preauth " option is selected when you create a new order. So, if you want to create different type, you must change it explicitly!
  • Click on "Save " Button
After you fill all required information you must save your newly created order. If the order is saved successfully you will see new created order in orders list.
 

Create new RatePAY Direct Debit Order

    • Click on „Plus“ icon on top right corner
    • New window will open to create a order
 
    • Select order Payment method type. (In this case RatePAY Direct debit)
 
    • New input fields will appear for creating a RatePAY Direct debit order.
    • First part are optional information input fields
      • MerchantOrder ID - Id of the order for merchant
      • Description  - Description of the order
      • Statement description - Statement description
    • Second part are fields with information about the Total amount of the order
      • Amount - Total amount of the order
      • Currency  - Currency of the order Important!:
  • Third part are input fields for bank account
      • Owner - Owner of the bank account
      • Bank name  - Name of the bank
      • IBAN (Required) - IBAN for bank account
      • BIC - BIC of the bank account
  • Next part is required Basket
  • To add basket Item you must click on "Add Item " button
 
  • It will open new section with basket additional fields.
  More about basket you can find in "How to manage Baskets ".  
  • Next part is "Customer information ", it contains section with customer information fields.
        • Email (Required)
        • Merchant customer Id
        • Additional Persona information
          • Title
          • First name (Required)
          • Last name (Required)
          • Phone
          • Fax
          • Mobile
          • Birthday (Required)
          • Gender
Important! - Be careful with the required fields, because otherwise you can't save the Order.
  • Next part is "Billing Address ".
This part contains information about Billing Address.
          • Title
          • First name (Required)
          • Last name (Required)
          • Street (Required)
          • Street Number (Required)
          • City (Required)
          • Zip (Required)
          • Country (Required)
          • State
          • Phone (Required)
          • Fax
          • Mobile
          • Additional details
Important! - Be careful with the required fields, because otherwise you can't save the Order.
  • Next part are type of the transaction action. There are two types of transaction actions for RatePAY Direct Debit: Preauth and Debit.
 
  • Last part for Accepting Terms and Privacy.  Both checkboxes are required.
   
  • Click on "Save" Button
After you fill all required information you must save your newly created order. If the order is saved successfully you will see new created order in orders list.
