  • Navigation to Customers
  • Create customer

Navigation

Customers information can be found in "Customers" menu item. After you click on menu item you must see customers list.
 

Create

Creating a customer is pretty simple, since there is only one value which is mandatory.
1 Add the customers E-Mail here, this field is mandatory.
2 You can add a custom customer ID here.
3 One or more addresses can be added here to the customers account.
4 You can add one or more personas to the customers account with this button.

Delete

If you want to delete customer you must select "Delete customer" button on the top right corner: You will see modal window with for confirmation of deleting: After you confirm, the customer will be deleted from the customers list.
