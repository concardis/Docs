Navigation

Payment methods information can be found in "Payment methods" menu item. After you click on menu item you must see payment methods list.
 

Payment method status

In the Payment methods table you can see the status of the payment method. When payment method is with "Active" status, you can see information about payment method configuration if you click on "Actions". When payment method is with "Inactive" status, if you click on "Actions" you will see error message on top of your window.
 

Credit card configuration settings

If credit card payment method is with "Active" status, by clicking on "Actions",  you will see all available information about credit card configuration. There you can find different sections with configuration information.
  • Card Acceptor information
  • Contracts information
  • Other credit card configuration informationAt the bottom of the page you will find : - Is Recurring verification enabled - Is CVC required - Is Cardcheck enabled
 

Paypal configuration settings

If paypal payment method is with "Active" status, by clicking on "Actions",  you will see all available information about paypal configuration. - First section: Paypal user, password and signature - Second: SFTP username and password - Third: Accepted currencies (More about currency codes...)
 

Paydirekt configuration settings

If paydirekt payment method is with "Active" status, by clicking on "Actions",  you will see all available information about paydirekt configuration. - First section: API key and API Secret - Second: Accepted currencies (More about currency codes...)
 

RatePAY (Direct Debit) configuration settings

If RatePAY Direct Debit payment method is with "Active" status, by clicking on "Actions",  you will see all available information about RatePAY configuration. - First section: Device Ident Ids - Second: Accepted currencies (More about currency codes...) - Third: Information about different RatePAY credentials.
 

RatePAY (Installment) configuration settings

If RatePAY Installment payment method is with "Active" status, by clicking on "Actions",  you will see all available information about RatePAY configuration. - First section: Device Ident Ids - Second: Accepted currencies (More about currency codes...) - Third: Information about different RatePAY credentials.
 

RatePAY (Invoice) configuration settings

SEPA configuration settings

If SEPA payment method is with "Active" status, by clicking on "Actions",  you will see all available information about SEPA configuration. - First section: Payout delay in days - Second: Accepted currencies (More about currency codes...)
