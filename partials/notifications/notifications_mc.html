Scope of Notification Service Subscription


When using Merchant Center for setting up notifications, merchants can subscribe to be notified for all or some events from the following four event categories:

  • Transaction Events
  • Customer Events
  • Payment Instrument Events
  • Order Events

More detailed information about events can be found under the following link: - Notifications in API



Management of Notification Service



Accessing 'Notifications' section

Management of notifications can be done when accessing "Notifications" section on Merchant Center menu



Creating new notifications


Step 1 - Clicking on „Plus“ icon in the top right corner
Outcome - A new window containing all parameters of the notification will be opened
Step 2 - Selecting notification type option (email for the current example)
Outcome - A new 'Email' field will be displayed
Step 3 - Filling information fields which individualize new notification (mandatory - email address receiver, optional - appropriate name and description)
Step 4 - Selecting events for which merchant wants to be notified
  • Option one - Selecting all available events - by marking 'Select All' checkbox
  • Option two - Selecting one or more events - by marking checkbox/es of the desired event/s

Important Note:

When creating a notification, by default, the "Activate notification" checkbox is marked . If the merchant wants to create an inactive notification, this checkbox has to be unmarked.


Step 5 - Saving the new notification - by pressing "Save" button on the bottom of the page
Outcome - Successfully created notification is displayed in the notification list


Actions with existing notifications



Reviewing notification details


All details for any existing notification from the notification list can be reviewed by the merchant by selecting dots in the column 'Actions' for a chosen notification.
After selecting the dots a new window will be opened displaying available details for the chosen notification.


Editing notifications


An 'Edit' button is present on the window containing details for the selected notification.
After pressing the 'Edit' button a new window will be opened displaying notification details that can be edited.

The following edit actions are possible:

  • Changing notification type - in the current example from email to URL

    • Important Note: Changing the notification type requires a change in the field below 'Email'/ 'URL' - in the current example a valid URL has to be specified.
  • Changing notification information

    • Email/URL field is mandatory - a valid email address/URL is required

    Name & Description fields are optional


  • Changing events for which merchant to be notified

  • Changing active status of the notification - when marking/unmarking field 'Activate Notification'

    • Important Note: In order to be applied all changes have to be saved by clicking on the 'Save' button.


Deleting Notifications


Deleting a chosen notification from the notifications list can be done by clicking on the delete icon placed at the end of notification row.
After clicking on the delete icon, a modal window with cancel and confirmation options will be displayed.
After pressing 'Delete' button on the modal window the selected notification will be deleted from the notifications list.


Filtering Notifications List


A list of notifications can be narrowed down by applying filters based on various criteria including; creation date, update date, active status and type of notification.

Clicking in the field 'Filters' above the notifications list will make all filters visible.




Filtering by "Created at"


There are three options for setting up date/s for the filter 'Created at".

For all three options desired date/s has/have to be chosen from the calendar by clicking on the calendar icon in front of the input field/s as shown below.


Option 1 - Specifying only the start date of a date range.

The end date of the range will not be specified and the second input field will remain empty.


Outcome - SAfter pressing 'Apply' button the list of notifications will be narrowed down to those notifications which are created from the specified start date to current date.
Option 2 - Specifying only the end date of a date range.

The start date of the range will not be specified and the first input field will remain empty.


Outcome: After pressing 'Apply' button the list of notifications will be narrowed down to all existing notifications which are created from the beginning to the specified end date.
Option 3 - Specifying both the start date and the end date of a date range.

Both input fields of 'Created at' filter will be filled.


Outcome: After pressing 'Apply' button the list of notifications will be narrowed down to all existing notifications which are created from the specified start date to the specified end date of the date range
Important Note: Deleting of already selected date/s is possible by pressing 'X' icon at the end of the second input field for dates.


Filtering by "Updated at"


Filter 'Updated at' works in a similar way to the way of working of filter 'Created at'

There are three options for setting up date/s for the filter 'Updated at".

For all three options desired date/s has/have to be chosen from the calendar by clicking on the calendar icon in front of the input field/s as shown below.


Option 1 - Specifying only the start date of a date range.

The end date of the range will not be specified and the second input field will remain empty.


Outcome - After pressing 'Apply' button the list of notifications will be narrowed down to those notifications which are updated from the specified start date to current date.
Option 2 - Specifying only the end date of a date range.

The start date of the range will not be specified and the first input field will remain empty.


Outcome: After pressing 'Apply' button the list of notifications will be narrowed down to all existing notifications which are updated from the beginning to the specified end date.
Option 3 - Specifying both the start date and the end date of a date range.
Outcome: After pressing 'Apply' button the list of notifications will be narrowed down to all existing notifications which are updated from the specified start date to the specified end date of the date range.
Important Note: Deleting of already selected date/s is possible by pressing 'X' icon at the end of the second input field for dates.


Filtering by "Active" status


Notifications with 'Active' status can be filtered from the full list of notifications by marking checkbox 'Active' in 'Filters' section and pressing button 'Apply'.


Outcome - After pressing 'Apply' button the full list of notifications will be narrowed down only to notifications which are with status 'Active'


Filtering by "Notification type"


Notifications with chosen notification type can be filtered from the full list of notifications by marking corresponding to the notification type checkbox in 'Filters' section and pressing button 'Apply'.


Outcome - After pressing 'Apply' button the full list of notifications will be narrowed down only to notifications from selected notification type.
